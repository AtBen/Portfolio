Mes interventions

  • Intégration de maquette

    Vous avez votre design, il ne reste plus qu'à le transformer en code pour l'implémenter dans votre projet.

  • Développement d'interfaces

    Plus qu'une simple intégration, vous avez besoin de développer l'interface dynamique avec des frameworks reconnus.

  • Mise à jour

    Votre projet a déjà été développé mais vous avez besoin d'apporter une correction ou d'ajouter une nouvelle fonctionnalité.

  • Projet complet

    La maquette est prête. Il reste le développement du site au complet (mise en place d'un CMS, développement de l'interface, mise en ligne).

Les compétences

  • HTML

    Un code HTML5 commenté et minimaliste pour un maximum d'efficacité.

  • CSS

    Du style maintenable grâce à une combinaison de SCSS et de BEM (convention de nommage).

  • Javascript

    Un peu de jQuery et AngularJS pour animer les interfaces de vos applications.

  • Design

    La gestion des designs (découpe, déclinaison, ...) de manière efficace grâce à Photoshop.

  • Versionning

    Suivi des avancées et conservation décentralisée du code grâce à Git.

  • Performance

    Optimisation du code (compilation, minimalisation) et des balises utilisées pour un résultat optimal.
